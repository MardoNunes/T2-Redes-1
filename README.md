### Jogo Black Jack em uma rede em anel
Trabalho desenvolvido na disciplina de Redes De Computadores 1, Universidade Federal do Paraná, por Mardoqueu F. Nunes.

O trabalho consiste na construção de uma rede em anel para implementar o jogo Black jack (21).

#### OBS
- Deve se iniciar em cada terminal com ID, endereço e porta de escuta.
- O jogo possui apenas uma rodada, se tem um ganhador ele encerra.
- Existe uma função de distribuição de carta que quebra os princípios de rede em anel!
